KeyCove Feature 2: Advanced Server-Side Search Engine

What was added:
- Backend search endpoint: GET /api/properties/search
- Server-side filters: price, beds, baths, square feet, property type, listing type, amenities, available before, posted after
- Sorting: newest, price low-high, price high-low, most beds, most baths, largest area, area A-Z
- Search query logging expanded in SearchQuery model
- Advanced filter UI added on Home page
- Active filter chips and sort dropdown added
- Property cards now show amenity previews

Before you run on your own computer:
1. Open the project root in VS Code or your terminal.
2. In BOTH client and server folders, run npm install.
   This is important because the zip contains Windows-installed node_modules and they can break on another machine/environment.
3. Make sure server/.env has your correct MongoDB Atlas URI.
4. Start server: npm run dev (inside server)
5. Start client: npm run dev (inside client)
6. Open the Vite local URL in your browser.
7. On the home page, click "Populate Dhaka Seed Data" if your database does not already contain the demo property data.
